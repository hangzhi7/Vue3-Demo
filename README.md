# Vue3-Demo简单的项目学习知识

## 1、项目介绍

一个简单的Vue3前端项目学习知识，从0到1搭建一个完整项目，给后续的老项目源码升级备用一些简单的知识模板

## 2、项目技术栈

- Vue3
- Vite
- Vue-Router
- Element-Plus
- Axios
- Less 或 SCSS
- Pinia

## 3、项目运行

```bash
# 1、克隆项目
git clone https://github.com/hangzhi7/Vue3-Demo.git

# 2、进入项目目录
cd Vue3-Demo

# 3、安装依赖
pnpm install

# 4、运行项目
pnpm run dev

# 5、打包项目
pnpm run build
```
